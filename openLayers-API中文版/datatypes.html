OpenLayers.Bounds

描述

此类的实例描述bounding boxes.含如left, bottom, right, top floats数据。所有值初始化为null,在用bounds前应该确保已经设置好。

属性

属性项

类型

含义

Number

最小的水平坐标

bottom

Number

最小的垂直坐标

right

Number

最大的水平坐标

top

Number

最大的垂直坐标

构造方法

OpenLayers. Bounds

函数声明

OpenLayers. Bounds (left,bottom,right,top)

参数

left-类型：Number，左边界。注意考虑宽度，理论上小于right值

bottom-类型：Number，下边界。考虑高度，理论大于top值

right-类型：Number，右边界

top-类型：Number，上边界
