弹出框

 

Popup

描述

一个弹出浮云是一个很小的div，在地图上可以打开或者关闭，通常情况下打开响应点击一个标记，弹出的浮云不要求他们自己有层并且不要求在地图上添加使用OpenLayers.Map.addPopup方法。

属性

属性项

类型

含义

autoSize

Boolean

自动调整弹出的内容

minSize

OpenLayers.Size

在弹出的内容里允许有最小的尺寸大小

maxSize

OpenLayers.Size

在弹出的内容里允许有最大的尺寸大小

panMapIfOutOfView

Boolean

当绘制时，地图的上下左右移动如同整个的弹出浮云是可见的

构造方法

OpenLayers.Popup

函数声明

OpenLayers.Popup(id,lonlat,contentSize,contentHTML,closeBox,closeBoxCallback)

参数

id-类型：String。弹出浮云的唯一标识，如果null是标识符，则将会自动生成

lonlat-类型：OpenLayers.LonLat，在地图上弹出的位置将会显示出来

contentSize-类型：OpenLayers.Size，内容的大小

contentHTML-类型：String，一个HTML字符串显示在弹出浮云的里面

closeBox-类型：Boolean，在弹出浮云的里面是否显示一个closeBox

closeBoxCallback-类型：Function，函数调用，并对closeBox点击

描述

创建一个弹出的浮云

返回值

无

getSafeContentSize

函数声明

getSafeContentSize:function(size)

参数

size-类型：OpenLayers.Size，想要弹出的尺寸大小

描述

弹出的这个大小既不是小于指定的最小尺寸，也不是大于最大的尺寸（计算相对于视口的大小）

返回值

类型：OpenLayers.Size
