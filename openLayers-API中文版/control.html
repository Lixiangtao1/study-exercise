OpenLayers.Control

描述

控件能影响地图的显示和操作。他们支持所有涉及平移和按比例尺缩放的操作，缺省控件被添加到所在的地图 ，通过传入div到参数项添加一个控件到一个外部的div

OpenLayers.Control.NavigationHistory

描述

导航历史记录控件，这是一个创建从属previous 和 next控件的元控件。调用previous 和 next控件去恢复先前的和紧接着的历史状态。previous 和 next控件在有能激活的状态要恢复时激活，当没能激活的状态要恢复时关闭。继承于OpenLayers.Control

属性

属性项

类型

含义

previous

OpenLayers.Control

被此控件管理的触发恢复先前状态管理的按钮类型控件

previousOptions

Object

next

OpenLayers.Control

恢复下一状态的按钮类型控件的触发管理控件

nextOptions

Object

在为next控件设置可选参数的构造器

limit

Integer

可选的保留历史项的数目的限制，如果为空，则无限制，默认为50.

OpenLayers.Control. OverviewMap

描述

创建一个缩略图显示主窗口图的扩展和提供附加的导航控件。用OpenLayers.Control.OverviewMap构造器创建一个新的缩略图。继承自OpenLayers.Control

属性

属性项

类型

含义

ovmap

OpenLayers.Map

缩略图自身的一个引用

size

OpenLayers.Size

屏幕缩略图尺寸

layers

Array(OpenLayers.Layer)

缩略图中安排好的图层序列

minRectSize

Integer

缩略图上矩形范围最小化的宽和高。当矩形范围达到此尺寸，他将被依据minRectDisplayClass属性的值所代替。缺省为15像素

minRectDisplayClass

String

达到minRectSize时矩形范围代替样式的类名，此字符串将被在显示类上添加后缀，缺省为“RectReplacement”

minRatio

Float

缩略图分辨率和主窗口地图分辨率的比例，在此基础上能更好缩小缩略图上的要素

maxRatio

Float

缩略图分辨率和主窗口地图分辨率的比例，在此基础上能更好放大缩略图上的要素

mapOptions

Object

缩略图的地图构造器的任何非缺省属性的对象

属性

属性项

类型

含义

multiple

Boolean

允许选择复合几何图形。缺省为false

clickout

Boolean

当点击要素以外的位置时取消选择要素。缺省为true

toggle

Boolean

单击已经被选择的要素时取消选择。缺省为false.只是指悬停时为false

hover

Boolean

鼠标在其上时选择，移开时取消选择。如果为true，将忽略单击只监听鼠标移动

box

Boolean

允许拉框选择

Function

Function

Array(String)
