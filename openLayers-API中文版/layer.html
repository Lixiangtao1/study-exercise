图层

 

Layer

描述

图层(Layer)是一切操作的基本,首先要创建一个基础层才能在图层上显示控件，操作控件等功能。

属性

属性项 类型 含义

Id

String

唯一标识，每个图层只有唯一的id属性值

Name

String

图层的名字

Div

DOMElement

图层包含在这个元素中

alwaysInRange

Boolean

布尔类型，如果一个图层显示的不是基本级别，

那么应该设置为true

Events

OpenLayers.Event

触发的事件，然后交由OpenLayers.Events类处理

Map

OpenLayers.Map

当这个图层添加到这个地图上时，设置变量，然后调用setMap方法

isBaseLayer

Boolean

布尔类型，判断是否是基础层，无论这个图层

是否是基础图层，他们都应该显示设置所有的

子类,默认为false

displayInLayerSwitcher

Boolean

布尔类型，显示图层的名字在图层开关器里面，

默认为true

visibility

Boolean

布尔类型，图层应该被显示在地图上，默认为

True

attribution

String

在OpenLayers.Control.Attribution被添加到地图上时候就显示

eventListeners

Object

如果设置一个选项在构造（初始化）阶段，那么这个事件的监听着对象将要注册在OpenLayers.Events

Gutter

Integer

忽略确定宽度（像素）的周边范围的图像瓦片

Units

String

图层的地图单位

Scales

Array

数组类型的地图大小

resolutions

Array

地图列表的分辨率

maxExtent

OpenLayers.Bounds

图层的最大范围

minExtent

OpenLayers.Bounds

图层的最小范围

maxResolution

Float

图层最大的分辨率，默认最大的是360度/250像素

minResolution

Float

图层最小的分辨率

numZoomLevels

Integer

缩放级别的大小

minScale

Float

最小的尺寸

maxScale

Float

最大的尺寸

displayOutsideMaxExtent

Boolean

布尔类型，判断请求地图瓦片是否完全超出了这个图层的最大范围

wrapDateLine

Boolean

布尔类型

transitionEffect

String

过渡效果的使用是在地图上下左右移动或者缩放的时候
