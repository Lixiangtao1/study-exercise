OpenLayers. String

描述

包含字符串处理功能。

属性

无

构造方法

无

方法

startsWith

函数声明

startsWith:function(str,sub)

参数

str-类型：String，需要测试的字符串。

sub-类型：String，匹配子串

描述

确定此实例的开头是否与指定的字符串匹配

返回值

类型：Boolean，第一个字符串是以第二个字符串（子串）开头时为真，否则为假

contains

函数声明

sontains:function(str,sub)

参数

str-类型：String，需要测试的字符串。

sub-类型：String，匹配子串

描述

确定此实例的开头是否与指定的字符串匹配

返回值

类型：Boolean，第一个字符串如果包含第二个字符串则为真，否则为假

trim

函数声明

trim:function(str)

参数

str-类型：String,含有空格符的字符串，此字符串本身没被修改

描述

确定此实例的开头是否与指定的字符串匹配

返回值

类型：Boolean，第一个字符串如果包含第二个字符串则为真，否则为假

camelize

函数声明

camelize:function(str)

参数

str-类型：String, 要转换格式的字符串，方法执行后原始字符串并没改变

描述

对含连字符（-）的字符串进行格式转换，如：“chicken-head”转换格式后为“chickenHead”， “-chicken-head” 转换格式后为“ChickenHead”

返回值

类型：String，返回转换好的新的字符串

format

函数声明

format:function(template,context,args)

参数

template-类型：String，一个要被替换“token”的字符串。一个临时的含将被context[“token”]值替换“token”的字符串

context-类型Object，一个匹配token格式的可选对象。如果没有传送context，将使用窗口对象。

args-类型：Array，传给context中任何函数的可选自变量。如一个context属性时一个函数，token将被含此自变量的函数返回值替换

描述

将$(token)利用上下文函数并带入参数计算的结果替代原标记token

返回值

类型：String，返回转换好的新的字符串

常量

无

OpenLayers.Array

描述

包含操作数组的简便方法。

属性

无

构造方法

无

方法

filter

函数声明

filter:function(array,callback,caller)

参数

array-类型：Array,需要过滤的数组。此数组没被改变。通过callback添加的元素将不能被访问。

callback-类型：Function，数组里每个元素调用的函数。如果函数返回true，元素将被包含在返回值中。带有三个自变量：数组中的元素、元素的索引、数组本身。可选的caller参数被指定为callback，作为caller的设置。.

caller-类型：Object，当调用callback时被设置的可选的对象。

描述

过滤数组。

返回值

类型：Array，callback为true时传入数组的元素组成的一个数组

常量

无
