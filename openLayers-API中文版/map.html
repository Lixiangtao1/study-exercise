地图对象

 

Map

描述

OpenLayers.Map的实例是内嵌于网页的交互式地图，可以在一个地图上增加相关控件（OpenLayers.Control）和加载相应图层(OpenLayers.Layer)

属性

属性项 类型 含义

Events

OpenLayers.Events

事件对象可以处理地图上的所有事件

div

DOMElement

地图所在的网页DOM元素，一般为div对象

layers

Array(OpenLayers.Layer)

地图中已加载的所有图层

baseLayer

OpenLayers.Layer

当前的基础图层

tileSize

OpenLayers.Size

地图默认的瓦片大小

units

String

地图单位

resolutions

Array(Float)

地图中能显示的分辩率（地图单位/像素）列表，以降序排列

maxResolution

Float

最大的分辨率，默认最高是360度/256像素，相当于缩放级别0

minResolution

Float

最小的分辨率

maxScale

Float

最大比例尺

minScale

Float

最小比例尺

maxExtent

OpenLayers.Bounds

地图的最大范围

minExtent

OpenLayers.Bounds

地图的最小范围

restrictedExtent

OpenLayers.Bounds

限制地图导航的范围

numZoomLevels

Integer

地图的缩放级别数

theme

String

地图的主题样式，相对路径的CSS文件

fallThrougth

Boolean

判断是否应该允许事件在地图上对下面的元素失效

eventListeners

Object

如果在构造方法中设置该选项，这个事件监听对象将要注册于OpenLayers.Events中

域值说明

Z_INDEX_BASE

Object

基本的Z-索引下标是不同的类

BaseLayer:基础层的下标，默认为100.

Overlay:覆盖层的下标，默认的为325.

Feature:要素的下标，默认的为725.

Popup:弹出浮云的下标，默认的为750.

Control:控件的下标，默认的为1000

EVENT_TYPES

Array(String)

支持应用事件的类型

addLayer:图层被添加时触发事件

removeLayer:图层被删除时触发事件

changeLayer:在改变图层名字，顺序改变，或者可见性改变之后触发事件

moveStart:开始拖拉，缩放之后触发事件

moveEnd:拖拉，缩放操作完成之后触发事件

Popupopen:弹出浮云打开之后触发事件

removeMarker:标记被删除时触发事件

addMarker:标记被增加时触发事件

drag:拖拉之后触发事件

mouseover:鼠标经过地图时触发事件

mouseout:鼠标移开地图时触发事件

TILE_WIDTH

Integer

默认为256瓦片的宽度

256:瓦片的宽度

TILE_HEIGHT

Integer

默认为256瓦片的长度

256:瓦片的长度
