Geometry.Collection

描述

Geometry.Collection是几何图形的集合，他们存储在本地的参数组件中。随着新的几何图形添加到集合中，他们不是克隆，当移动几何 图形时，他们需要指定引用。 这个getArea和getLength函数只能通过遍历组件，总计他们各自的面积和长度。创建一个新的实例要构造 OpenLayers.Geometry.Collection。继承自OpenLayers.Geometry

Geometry.LinearRing

描述

一个线性环是一个特殊的封闭的线串。它会自动关闭在每个addPoint/removePoint，用来添加一个复制的第一个点作为最后的一个点.。继承于OpenLayers.Geometry.LineString

属性

无

构造方法

OpenLayers.Geometry.LinearRing

函数声明

OpenLayers.Geometry.LinearRing(points)

参数

Points-类型：Array(OpenLayers.Geometry.Point)，点对象

描述

线性环构造了一个数组类型的点。这个数组可以表示一个关闭或者打开的环，如果这个环打开了，这个构造方法将要关闭这个环

返回值

无

addComponent:function(point,index)

参数

Point-类型：OpenLayers.Geometry.Point，点对象  

Index-类型：Integer，插入组件数组的下标

描述

添加一个点到几何图形组件，如果这个点添加到组件数组的末端，并且相同的点作为最末端的点已经在数组中，重复的一点是不能被添加的，这已经影响了关闭的环

返回值
