标注

 

Icon

描述

图标表示在屏幕上的图形图标，通常使用结合OpenLayers.Marker类在屏幕上表示标记，一个图标有一个url,大小和位置。也包含一个偏移量允许中心点正确的表示，可以提供作为一个合适的偏移量，也可以提供一个函数计算期望的偏移量。

属性

无

构造方法

OpenLayers.Icon

函数声明

OpenLayers.Icon(url,size,offset,calculateOffset)

参数

url-类型：String，图标的url路径

size-类型：OpenLayers.Size，图标的大小

offset-类型：OpenLayers.Pixel，图标的偏移量

calculateOffset-类型：Function，计算的偏移量

描述

创建一个图标，在div中是一个图像的标签

返回值

无

方法

无

常量

无

OpenLayers.Marker(lonlat,icon)
