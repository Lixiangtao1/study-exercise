Feature.WFS

描述

WFS作为一个要素类在WFS图层上控制“点”的WFS类型。继承于OpenLayers.Feature

OpenLayers.Feature.WFS(options,layer)

参数

Options-类型：Object,可选的对象

Layer-类型：OpenLayers.Layer,图层对象

描述

创建一个格式化的WFS-T。这就要求一个图层应该有两个属性：几何图形列和类型名字。解析这个格式完全由GML的子类来完成:这里只有一个记录器，此记录器使用了大量的GML图层代码，并包装在事务处理元素里
